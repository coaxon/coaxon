# CoAxon E-commerce OS

Welcome to the CoAxon ecosystem. CoAxon is an advanced, cross-platform E-commerce Operating System designed for professional sellers and modern retail operators.

Our platform orchestrates your entire supply chain and multi-channel sales footprint into a single, unified decision surface.

## The CoAxon Ecosystem

CoAxon integrates natively with major platforms to provide global visibility and unified controls:

- **[Amazon MCP](https://github.com/coaxon/amazon-mcp)** — Open-source Model Context Protocol server for Amazon SP-API and Advertising API. Handles the Amazon layer.
- **Meli MCP (Coming Soon)** — Connects Mercado Libre (Full).
- **TikTok MCP (Coming Soon)** — Connects TikTok Shop.

## Key Capabilities

By connecting the fragmented e-commerce landscape, the CoAxon Command Center enables:

* **Global Inventory Pooling** — Unified view across FBA, Meli Full, and TikTok warehouses.
* **Cross-Platform Price Parity** — Intelligent pricing engine that prevents TikTok flash sales from triggering Amazon Buy Box loss.
* **Unified Daily Briefing** — One Slack report, three platforms, one decision surface.

## Architecture & Command Center

While individual platform nodes (like `amazon-mcp`) operate as open-source MCP servers for seamless AI integration, the broader **CoAxon Command Center** acts as the central brain. 

The Command Center (Pro) handles cross-domain rules, global orchestration, and multi-tenant management. It is designed for scale, security, and true operational intelligence, never exposing your proprietary strategies to public repositories.
